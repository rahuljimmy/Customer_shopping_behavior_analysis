# 🛍️ Customer Shopping Behavior Analysis (Python + SQL + Power BI) Project

**An End-to-End Data Analytics Project** using **Python, PostgreSQL & Power BI** to analyze customer purchasing behavior, uncover business insights, and support data-driven decision making.

---

## 📌 Project Overview

This project analyzes **3,900 customer shopping transactions** to identify purchasing trends, customer segments, product performance, subscription behavior, and revenue patterns. The complete analytics workflow covers **data cleaning, feature engineering, SQL business analysis, interactive dashboard creation, and business recommendations**.

* Customer spending patterns across demographics
* Product & category performance analysis
* Subscription vs. Non-subscription comparison
* Customer segmentation based on purchase history
* Revenue analysis by age group, gender, and shipping type
